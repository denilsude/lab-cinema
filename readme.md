Resumo dos Passos para Testar Agora
Corrija o arquivo LancheForm/index.tsx com o código acima.

No Terminal 1 (Backend):

cd frontend

npm install (espere terminar)

npm run server

No Terminal 2 (Frontend):

cd frontend

npm run dev

Abra o navegador em http://localhost:5173.

Se a tela continuar branca, aperte F12, vá na aba Console e me mande o erro vermelho que aparecer lá.
